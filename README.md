Face-Recognization-Access-Control-System
This is our Face-Recognization-Access-Control-System project
- This system includes 3 parts:
  + ESP32 CAM: the eye of the system to see the user's face
  + STM32F411VET6: open the door after recognize face and display words on LCD16x2.
  + The web server: display user's information, sign up new user's informationa and save it in firebase.
